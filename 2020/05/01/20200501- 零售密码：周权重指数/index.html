零售密码：周权重指数

零售业的规律是以周为单位不断循环的过程。无论是传统零售还是电子商务都会遵循这种规律，区别是传统零售一般周末是销售高峰，电子商务周末反而一般。

权重指数的计算

为了分析零售业的日销售规律，我们构造权重指数来进行观察，权重指数的目的是为了给365天都赋予权重值。正常状态下赋予每天常规的权重，促销时则使用将促销级别考虑进去的特殊权重，节假日则使用节假日权重。

正常状态下的权重指数计算：

  1. 计算企业周权重指数：

    • 收集企业每个完整店铺最近一个完整年度的日销售额数据
    • 数据预处理：剔除法定节假日、法定假期的调休日、行业特殊日（如情人节、三八节、儿童节、圣诞节等）、非正常销售日（指不可抗力或人为影响的销售日）
    • 以周为单位整理数据，计算星期N 的平均销售额
    • 找到一周中最低的平均销售额，设定它的日权重指数为1.0，然后分布用其余六天的平均销售额除以这个最小值，得到其余六天的日权重指数
    • 将7天的日权重指数相加，即是企业的周权重指数

  2. 计算各分部（分店）的日权重指数：

    • 收集分部三个月的销售数据，一般是最近两个月和去年同期月份数据。如预测2013年10月的分部销售规律时，可以收集2013年8月、9月和2012年10月的数据。
    • 数据预处理：同上
    • 以周为单位整理数据，计算星期N 的平均销售额和平均周销售额。
    • 星期N 的日权重指数 = （ 星期N 的平均销售额 ➗ 平均周销售额 ） ✖ 企业周权重指数

节假日的权重指数计算：

  1. 取三年历史数据，每年取节假日附近2~4个标准周的销售数据作为参考
  2. 计算附近2~4个标准周的单位权重（销售值）值，计算公式为：总销售额 ➗ 总权重指数
  3. 计算节假日的权重指数：日权重指数 = 日销售额 / 单位权重（销售）值
  4. 重复以上计算过程，计算三年的节假日权重指数，取三年平均。

促销日的权重指数计算：

日权重指数 = 正常情况下的日权重指数 * （1 + 该促销活动历史增长率 ）

权重指数的应用

判断零售店铺周销售规律

权重指数可以用来分析店铺、区域、品牌等等对象的销售规律，对品牌或企业营运有积极的指导意义，我们可以根据权重指数的高低来安排店铺的员工排班、安排店铺的陈列调整时间、部署送货时间窗口等。

分解日销售目标

日销售目标 = 月销售目标 ✖ （日权重指数 ➗ 月权重指数 ）

月销售预测

用1-18日的实际销售额预测月销售额

月销售预测值 = 1-18日总销售额 ➗ （1-18日总日权重指数 ➗ 月权重指数 ）

注意：用于预测的天数需从1日开始，至少大于10天，且每天都可以实时更新预测数据，所用预测天数越多越准确。

销售对比

进行对比分析的第一要素就是必须要有可对比性，否则就可能误导受众。日销售数据的同比需要遵循如下准则：

  1. 首先遵循星期几对比星期几的原则，因为这是零售业的基本规律。
  2. 其次遵循公众假期对比原则，例如2014年5月1日（星期四）应该和2013年4月29日（星期三）对比，因为这两个日子都是五一放假的第一天。
  3. 再次遵循中国阴历对等对比原则，这种情况只是出现在每年春季期间，初一和初一对比。春节放假结束后又回归到 1 中对比。
  4. 最后遵循阳历对等对比原则，这适合一些特殊日期和特殊的行业，例如鲜花和巧克力要遵循2月14日对比原则，女士用品特别是服装和化妆品遵循三八节对比原则，玩具行业遵循六一对比原则等。

如何解决上述 “时间标准” 有时没有对比性原则的问题？

可以使用单位权重值法。

单位权重（销售）值 = 某时间段总日销售额 ➗ 某时间段总日权重指数

单位权重（销售）值这个概念可以被放大，只需要把 “销售” 替换成例如店铺的人流、网站的UV（独立访客）、呼叫中心的电话量等就行。

神奇的黄氏曲线——单位权重（销售）值曲线

下面简称：权重曲线。

如果一个零售店铺的每日销售额是绝对服从周权重指数的规律，那对应的权重曲线则将是一条绝对水平直线，而这种情况是根本不可能出现的，正常的权重曲线是一条围绕某个值变化的曲线，正是这种变化给我们提供了去洞悉某些运营现象的可能。（有变化才有可能，有变化才有希望。）

这种变化说明有某种业务逻辑暗含其中，具体原因需要把权重曲线剥开看。

应用在销售追踪过程中

用权重曲线图监控销售变化，如发现 ”月初放松“ ，” 月末刹车“ 等现象。

特殊事件的量化处理——事后分析

如何评估 “某特殊事件” （如节日、促销、恶劣天气等）对销售的影响？

针对这类问题，我们可以借助权重曲线来进行评估。

绘制黄氏曲线：计算事件前一段时间的平均单位权重值、事件发生期间的平均单位权重值、事件发生后一段事件的平均单位权重值。

如何量化影响了多少销售额呢？

对于黄氏曲线的应用，一定需要注意是 “突发” 状态，就是要有非常清洗的时间节点、非常明确的事件信息状态才可以使用；另外建议要同时画权重曲线和黄氏曲线，权重曲线是用来进行追踪跟进的，用它可以及时反应状态在数据上的变化，以便及时采取对应的策略调整。黄氏曲线是用来分析和评估的，所以它是 “事后” 工作。
