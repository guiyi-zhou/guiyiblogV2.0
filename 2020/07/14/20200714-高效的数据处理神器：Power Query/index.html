高效的数据处理神器：Power Query

什么是Excel Power Query？

Excel Power Query 是Excel 用于数据分析的三大组件之一，其它两个分别是Excel Power Pivot 和 Excel Power View。在微软Excel2010 版和 Excel2013 版中作为插件使用，需要在官网进行下载，直到Excel 2016 版之后微软便将其作为 Excel 的组成部分嵌入在软件当中。

使用 Excel2010 版和 Excel2013 版的小伙伴可以到微软官网下载。

Excel Power Query插件下载地址：

https://www.microsoft.com/zh-CN/download/details.aspx?id=39379

（建议大家使用 Excel2016 专业版及以上版本，使用起来更稳定，能避免许多软件带来的小Bug）

Excel Power Query 主要用于获取和转换数据，对不同源的数据进行整理加工。Power Query 的操作对象是表，使用 Power Query 能帮助我们提高对表结构数据的处理速度，提高办公效率。

如何进入Excel Power Query？

那么如何进入Excel Power Query界面呢？

下面我们先来看一下Excel2010版和Excel2013版如何加载和进入Excel Power Query平台。

  1. 在官网下载Excel Power Query插件；
  2. 打开Excel COM加载项，勾选Excel Power Query组件；

  1. 任意选中一个空单元格，点击【从表】便可进入Excel Power Query 界面：

Excel2016专业版及以上版本中，Excel Power Query默认在数据选项卡下的【获取和转换】功能组中，点击【从表格】便可进入Excel Power Query界面：
